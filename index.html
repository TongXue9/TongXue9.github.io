Analog IC Building Blocks in CMOS (SPICE-Based Design and Simulation)

Designed and analyzed fundamental analog CMOS building blocks using SPICE-based circuit simulation, focusing on transistor-level behavior and core principles of analog IC design.

The project aims to build intuition for how analog integrated circuits operate at the device level, emphasizing the relationship between circuit topology and key performance metrics such as gain, bandwidth, linearity, and output impedance.

Core modules include current mirrors, differential pairs, and a basic two-stage operational amplifier. Each block is evaluated through DC, transient, and AC analysis to understand trade-offs in analog circuit design.
